## Open-source friendly

### 3D models/scenes

I [Tweeted a while back](https://twitter.com/joedavisdev/status/645654172503707648) asking for help finding 3D artwork with open-source compatible licences. Here's the list so far:

* [SketchFab](https://sketchfab.com/)
* [Morgan McGuire's Graphics Data](http://graphics.williams.edu/data/meshes.xml)
* [PixelLab](https://www.thepixellab.net/freebies)
* [Keenan Crane](http://www.cs.columbia.edu/~keenan/Projects/ModelRepository/)
* [Blend Swap](http://www.blendswap.com)
* [opengameart.org](http://opengameart.org)
